Dashboard - Nodejs, Express and Mongodb

Created a flowchart dashboard, which observing the petrol and diesel price in Bangalore.

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 
Prerequisites
1.Mongodb
2.Node js
3.Express js
3.npm
4.Terminal
5.Robomongo

Steps:-
1. Install Node, Express, Mongodb and npm.
2. Download the codes from github and do npm install.
3. Enter the data in to Mongodb through Robomongo or Terminal.
4. Now start the server on terminal (node server.js).
5. Start the app locally http://localhost:3300/.
